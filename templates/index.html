Learning with games

This website is the result of a scientific project focused on the ability of a computer to learn from humans, as they collaborate with the research just by playing games and having fun. The project is based on an approach called Games with a Purpose, that will be explored together with machine learning algorithms, in an attempt to expand the knowledge base of a never-ending language learner called NeLL.

You can read more about this project in the Project Overview section.
