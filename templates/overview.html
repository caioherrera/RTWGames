Project Overview

The main goal of this project is to expand the knowledge base of the computer system called Never-Ending Language Learner (NeLL), created as a research project at Carnegie Mellon University, allowing users to interact with the knowledge base, through an approach known as Games with a Purpose, explored by Luis von Ahn. Through a few games, users provide NeLL a wide range of candidates for entities and categories. After that, machine learning algorithms will be applied on the collected data, in order to directly improve the ontology and knowledge base of NeLL.

Click here to learn more about the NeLL project.

Until now, it is possible to access the data collected with this project using the /data page, where the user can filter parameters like the expression they are looking for, if it occurs as an entity or a category, and from which game the feedback came from. The generated data is presented in JSON format, so it's easier to understand and manipulate.

The future updates of this project will be mentioned here.
